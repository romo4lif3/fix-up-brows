Browsers Ranked


Safari

#3
Pros:

-Speedy load time
-Auto updates
-Parental Controls

Security tools:

-Pop-up Blocker
-Antispyware
-Antivirus
-Antiphishing
-Private Mode

Cons:

-Only available version on windows is
5.1.7(2012 version)
12.0.2 current on Mac Os
-Low customizability
-Issues with google as default browser
-Vulnerable to malware


Microsoft Edge

#2
Pros:

-Fast and simple interface
-Has Pdf and Ebook reader
-Search with cortana
-Tab organizer tools

Cons:

-fewer extension add ons
-No history search
-limited amount of accessible sites
-No drag and drop folder linking


Google Chrome

#1
Pros:

-Over 11,500 extensions
-Secure Searching
-Virus site warnings
-Developer tools
(redux tools, json and xml formatting tools)
-Customizable interface

Cons:

-Too many extensions slows browser performance
-Not the fastest browser
-Slower navigation
-Ad tracking



Personal Favorite

My personal favorite is google chrome for that its fast, simple and easy to use. As a developer I couldnt imagine having to use any other browser to test websites, find bugs, format data and use the extension tools in development. I rank it 5 out of 5 stars.
